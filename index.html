AI Solution Programmer

Build AI-powered automotive tools – from log analysers and compliance copilots to cockpit and EV analytics dashboards and interactive web portals.

Experience: 3–8 years Location: Bengaluru / Hybrid Type: Full-time
You will
  • Design and implement AI-backed tools for ADAS, E&E, cockpit and EV data.
  • Work with logs, specs and AIS / NCAP / ISO standards to build practical copilots.
  • Develop backend services, data pipelines and APIs for analytics & visualisation.
  • Prototype portals and dashboards (HTML / JS / React / Angular-based) that OEM teams can run locally or on cloud.
  • Collaborate directly with founder / domain experts to translate ideas into code quickly.
You bring
  • Strong Python skills (data processing, APIs, automation; ML experience is a plus).
  • Experience with modern web technologies such as React.js and/or Angular (bonus: Vue, TypeScript) for building interactive UIs.
  • Comfort with REST / JSON APIs and integrating AI backends into web front-ends (dashboards, portals, configuration tools).
  • Good understanding of at least one area: ADAS logs, CAN traces, BMS data, Android logs, etc.
  • Comfort reading technical specs / standards and turning them into structured rules / checks.
  • Mindset to ship small, high-quality tools fast rather than big “perfect” platforms.

How to apply: Send a short mail with links (GitHub / sample scripts / tools or portals you’ve built) and a few lines on your favourite automotive or web+AI problem to careers@ZYFNYXautolabs.com

AI Solution Tester

Own the quality of AI-driven tools – from test design and data selection to edge-case hunting across ADAS, E&E and EV analytics.

Experience: 3–8 years Location: Bengaluru / Hybrid Type: Full-time

  • Recreate real-world issues from OEM / Tier-1 feedback and convert them into regression suites.

  • Curiosity to challenge AI outputs, not just check if the UI “works”.
